# ITI Task Exam System

## Description

A simple online quiz application built with HTML, CSS, and JavaScript.
Users can sign up, log in, choose quiz settings (category, difficulty, and number of questions), take a timed quiz, and view results instantly.

## Features

- User signup and login
- Quiz settings page before exam start
- Questions fetched from Open Trivia DB API
- Timed quiz with progress tracking
- Instant score and answers review
- Data saved in browser localStorage

## Technologies

- HTML5
- CSS3
- JavaScript (ES Modules)
- Open Trivia DB API
- localStorage
